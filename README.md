# 📝 Priorito - Aplicación para gestión de tareas

Aplicación móvil desarrollada con **Ionic Angular + Cordova**, como parte de una prueba técnica para Accenture.  
El proyecto permite la gestión de tareas y clasificación por categorías, almacenamiento local y control de funcionalidades vía Firebase Remote Config.

## ✨ Características

- Creación, edición y eliminación de tareas
- Gestión de categorías relacionadas
- Filtro por categorías
- Almacenamiento local optimizado con SQLite
- Firebase Remote Config para feature flags
- Desempeño optimizado y arquitectura modular

## 📱 Tecnologías

- Ionic 7 + Angular
- Cordova
- Ionic Storage + SQLite
- Firebase Remote Config
- Git y GitHub para control de versiones
